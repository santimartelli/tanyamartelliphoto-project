Module: TheNavigation

Componente que muestra la barra de navegación para los visitantes o para el administrador si está fuera del panel de administración.

Data

Name Type Default value Description
scrolledNav boolean - Devuelve true si el usuario ha hecho scroll en la página.
mobileNav boolean - Devuelve true si el menú de navegación está abierto en dispositivos móviles.
scrollPosition number - Devuelve la posición del scroll en la página.
showDropdown boolean - Devuelve true si el usuario hace hover sobre el botón de portafolio.

Computed

Name Type Description
mobile boolean Devuelve true si el ancho de la pantalla es menor a 768px.
segundoNav boolean Devuelve true si el ancho de la pantalla es menor a 1235px.
categories Array Devuelve un array con las categorías de las fotografías.
isLoggedIn boolean Devuelve true si el usuario está autenticado.
