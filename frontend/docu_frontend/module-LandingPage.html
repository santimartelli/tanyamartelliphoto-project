Module: LandingPage

Componente que muestra la página de inicio.

Data

Name Type Default value Description
modules Array - Array con los módulos de Swiper.
currentImageIndex number - Índice de la imagen actual.
imagenesPortada Array - Array con las imágenes de portada.

Computed

Name Type Description
screenWidth number Devuelve el ancho de la pantalla.
mobile boolean Indica si la pantalla es móvil o no.
pictures Array Devuelve un array con todas las imágenes.
uniquePictures Array Devuelve un array con una imagen de cada categpría.

Methods

changeImage()

Cambia la imagen principal que que se muestra.
